## 🛡️ Password Security Tips

1. **Length over Complexity** - 16+ characters is better than short complex ones
2. **Use Unique Passwords** - Never reuse passwords across sites
3. **Use a Password Manager** - Bitwarden, KeePass, 1Password
4. **Enable 2FA** - Two-factor authentication wherever possible
5. **Never Share Passwords** - Legitimate services never ask for your password

| Entropy (bits) | Strength | Crack Time |

| 28-35 | Weak | Minutes to Hours |

| 60-127 | Strong | Centuries |
